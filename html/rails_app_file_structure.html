Basic Structure of a Rails 4 App

When you first create a Rails application by running "rails new app_name" from the command line, Rails generates the following file structure automatically.

  • app
    • assets
      • images
      • javascripts
        • application.js
      • stylesheets
        • application.css
    • controllers
      • application_controller.rb
    • models
    • views
      • layouts
        • application.html.erb
        • _errors.html.erb
  • bin
    • bundle
    • rails
    • rake
  • config
    • application.rb
    • database.yml
  • db
  • lib
  • log
  • public
  • spec
    • factories
    • features
  • test
  • tmp
  • vendor
  • Gemfile
  • Gemfile.lock
  • Rakefile
  • README.md
  • .gitignore
